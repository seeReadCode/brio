# brio
> n. a quality of being active, spirited, alive, and vigorous.

<https://brio.seeread.info>

## 🏁 Setup
```sh
bundle install
```

## 🏗 Dev
```sh
bundle exec jekyll serve
```

Add tags in `_tags/` and in the prose block of `_config.yml`.
